====
概览
====

`MoProfiler`_ 是一个综合性能分析工具，支持内存分析、时间分析、秒表打点等功能。
可以有效量化 Python 代码的性能，从而找到性能瓶颈所在，使性能优化可以有的放矢。

安装方法
========

您可以通过 ``pip`` 进行安装，本包兼容 ``Python 2.7`` & ``Python 3.5`` ::

    pip install moprofiler
